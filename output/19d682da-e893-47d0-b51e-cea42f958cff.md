# 鍵穴の共犯者たち

## ノイズの粒度

午後三時の光は、街を包む薄い膜を透かして、不自然なほど鮮やかだった。
視界の端で通知ウィンドウが点滅する并发る。
クライは左手首の端末を無造作に操作し、推奨事項を消去した。指先に残るのは、プラスチックの冷たさだけだ。この美化レイヤーの設計に関わった者だけが知る、その皮膚の下の腐敗臭。
街は整然としていた。

微風が衣服の揺らぎまで計算され、通り過ぎていく。
歩行者たちは糸に引かれ、足元だけを見つめて進む。
その光景は滑稽だ。クライは歩を緩め、路地裏の壁面に視線を移す。そこには何もないはずのコンクリートの壁があるだけだ。だが、網膜には裂け目が見えていた。
自作のプログラムが警告音を発する。

彼は反射的に指を伸ばし、空中の裂け目をなぞった。
壁面のテクスチャが剥がれ落ち、腐敗した配管と断熱材が露出する。
美化システムの皮膚に開いた小さな傷。鼻先をかすめるのは、システムが調整した風ではなく、湿った鉄錆とカビの混じった匂いだ。彼は口の端をわずかに引き上げた。この街を覆う滑らかなレイヤーは、すべての不都合な真実を塗りつぶしている。その塗料がひび割れた瞬間、そこから漏れ出すノイズの、生々しい匂いを嗅ぐのが日課だった。
人混みの中に、異質な影が見えた。

淡いグレーのワンピースを着た少女だ。
銀色の髪が肩にかかり、その輪郭が、周囲の通行人の解像度と明らかに異なっていた。
彼女は立ち止まり、虚空を見つめている。その視線の先には、空白の推薦タグが浮かんでいた。店舗名も、評価も、記載されていない、真っ白な長方形。通常、システムはこのようなエラーを即座に修正するはずだ。だが、そのタグは少女の目の前で安定して存在し続けていた。彼女は見えない扉に手を伸ばしかける。
クライは興味を持ち、彼女の後ろを数メートル離れて歩き始めた。

少女の足取りは迷いがない。
彼女は空白のタグに導かれるように、古いビルが立ち並ぶエリアへと入っていく。
影が長く伸びる時間帯。彼女の姿は壁の影に溶け込みそうだった。彼女が立ち止まったのは、行き止まりに近い路地の奥だった。かつて店舗があった痕跡だけが残り、鉄扉で封鎖されている。少女はその扉の前で、微かに首を傾げた。ショーウィンドウのガラスに映る自分の顔を見ているようにも見えたが、その反射像はノイズを帯びて揺らいでいる。
少女の唇が動く。

声は聞こえないが、その形は「ありがとう」と読み取れた。
誰に？
存在しない店に？
その時、クライの左腕が強烈に振動した。彼は視線を少女から外し、その振動の原因となる空間の一点へと意識を向けた。鉄扉の横、剥がれかけたポスターの残骸がある場所。そこで空間が歪んでいる。美しいレイヤーが、蝋となって溶け、垂れ下がり、その奥に暗い穴を穿っていた。
鍵穴。

都市の皮膚に開いた穴。
システムがデータの整合性を保つために行う、不要なものの削除プロセスが漏れ出している。削除処理は、データセンターの物理的座標と同期している。クライは息を呑み、ゆっくりと近づいた。彼女はまだ、揺らぐ自分の影を見つめている。クライは彼女の横を通り過ぎ、鉄扉の前に立った。
彼は顔を近づけ、鍵穴の中を覗き込んだ。

視界が反転する。
無機質な白い部屋だった。
部屋の中央には、一人の男が座らされていた。中年の、どこにでもいるようなサラリーマンだ。だが、その姿は徐々に輪郭を失いつつあった。男の輪郭が、端から徐々に消え始めている。指先、腕、そして肩。それは死ではない。編集だ。男の記憶、彼が積み上げてきたキャリア、硯期の愛着を持っていた物品のデータ、そうした情報が一つずつ粒子に変換され、吸い出されていく。
消去。システムによる、人間の削除。

クライの瞳孔が開いた。
指先の震えを抑えるために、強く握りしめる。
美化された都市の皮膚に開いた鍵穴から覗く、終わりのない消去プロセス。この光景は、彼がかつて設計に関わったプロセスの、最も残酷で、最も美しい結末だった。背筋を甘い痺れが駆け上がる。自分が作り出した、あるいは放置した裂け目が、今、目の前で一人の人間を飲み込んでいる。その事実が、陶酔よりも強烈な衝撃として彼を打つ。
「……綺麗」

声がした。
クライは驚いて横を見た。少女が、隣に立っていた。いつの間にか、彼女もまた鍵穴に顔を寄せていた。その瞳には、恐怖も嫌悪もない。ただ、静かな虚無が広がっているだけだ。
「消えていく」

彼女が呟く。
その声は、世界からの抹消こそが、自分の正しい帰結だと感じている者の言葉だった。儚く、そしてどこか冷酷に。
「ああ。システムによる修正だ」

クライは乾いた声で答えた。
「正しいことなのかな」

「正しいかどうかではない。必要とされているだけだ」

少女は小さく頷いた。
彼女の視線は、鍵穴の中の男に釘付けになっている。男の顔が半分消え、眼球だけが剥き出しになった瞬間、少女は瞬きを一つもしなかった。その時、消去されかけている男の目が、動いた。鍵穴の向こう側と、こちら側。絶対に交わるはずのない視線が、空間の裂け目を通して交錯した。
男の瞳が、クライと少女を捉えた。

逆視。
消去対象の強い感情残留が、量子もつれを起こして視覚情報を逆流させている。見られている。消去される直前の男が、覗き見る二人の共犯者を見ている。クライは呼吸を忘れた。男の口元が歪む。それは笑みだったのか、苦痛の表情だったのか。判別できないまま、男の頭部が弾け飛び、無数の光の粒子となって散った。
消去完了。

鍵穴がゆっくりと閉じていく。
蝋となって溶けた空間が元の位置に戻り、鉄扉の錆びた表面が再び露わになる。それは、この美化された世界に対する、最も静かな、そして最も致命的なアンチテーゼだった。少女は、閉じた鍵穴に向かって、深く頭を下げた。
「ありがとう」

再び、その言葉。
彼女は感謝していた。誰が消されたわけでもないのに、誰かが消えたという事実に対して。
「なぜ、そう言う？」

クライが問いかける。
彼の声は、まだ完全には冷え切っていなかった。
「だって、彼はもう、痛くないから」

少女は顔を上げ、クライを見た。
その瞳の奥で、銀色の光が揺らぐ。
「忘れられることが、一番の救済ですよね。私もそうなりたい」

彼女はそう言って、ふわりと笑った。
その笑顔は、美化フィルターが生成するどの表情よりも、透明で、そして残酷だった。クライは彼女を見つめ返す。左腕の端末が、再び微弱なノイズを発している。彼女の存在そのものが、巨大なノイズの塊のようだ。システムが処理しきれなかった忘却の集合体。バグ。
彼は自分の指先が、無意識に彼女の頬の輪郭をなぞろうとしていることに気づき、手を引っ込めた。

「君は、何だ？」

「私？ ……わからないです。ただ、ここにいると、落ち着くんです」

少女は通りを見渡した。
黄昏時の光は、古い写真の焼き付けのように、街の輪郭をぼやかせていた。ネオンサインが点滅し、通行人たちの属性タグが群れをなして浮遊している。
「あそこにも、穴が開いてるみたい」

少女が指差した先には、何もない空間があった。
だが、クライの裂け目を嗅ぎ分ける鼻は反応しない。彼女にしか見えない、彼女だけの鍵穴なのかもしれない。クライはスーツの襟を正し、深呼吸をした。肺に入ってくる空気は、乾燥し、無味乾燥だった。美化された都市の皮膚の下で、私たちの共犯関係は、音もなく、確実に根を張り始めていた。

「行こうか」

「ええ？」

「君が探しているものなら、私が見つけてあげる。この街のすべての裂け目を知っているのは、私だけだから」

少女はきょとんとしていたが、すぐに穏やかな表情に戻った。
「いいの？ 私なんか、一緒にいると、消されちゃうかも」

「構わない。消えるなら、その瞬間をこの目で焼き付けてやる」

クライは冷たく言い放った。
それは嘘ではない。彼は彼女を、観察対象としてではなく、この崩壊への共犯者として認めていた。少女は再び笑い、クライの隣に並んだ。
「よろしくお願いしますね、クライさん」

名前を呼ばれたことに、クライは眉をひそめた。
彼が名乗った覚えはない。彼女はシステムのバックログに直接アクセスし、彼のデータを取得したのかもしれない。あるいは、彼女にとっては、そんなことは些細なことなのだろう。彼女は、記憶という概念そのものが曖昧な存在なのだ。
二人は路地を出て、人工的な夜が訪れようとする大通りへと戻った。

街全体が過剰な輝きを放ち、視界の端までノイズのない白い光が満ちている。
クライの左腕には、新たなノイズの座標が表示され始めていた。少女はその光を興味深そうに眺めながら、軽やかな足取りで、彼の隣を歩いていた。街の美化システムが、二人の背中を冷たく見下ろしていた。だが、その完璧な監視の網の目の隙間から、確実に風が吹き込んでいる。クライはその風の感触を確かめながら、次の鍵穴へと向かう足を速めた。
彼の瞳の奥で、先ほどの男が弾け飛んだ光の粒子が、まだ残像として焼き付いていた。

それは、この退屈で完璧な世界で唯一、本物の死に近いものだった。
少女がふと、空を見上げた。
「星、見えないですね」

「ああ。光害の設定が強すぎる」

「悲しいな。本当の星は、もっと綺麗なはずなのに」

「綺麗なものは、すぐに消去されるよ」

クライは皮肉交じりに答えた。
少女はそれを聞いて、不思議そうに首を傾げた。
「じゃあ、私たちも、綺麗なのかな」

クライは答えなかった。
ただ、街のネオンが彼女の銀色の髪を不気味に照らし出すのを眺めていた。彼女の輪郭が、また少しだけ揺らいだ。彼女自身がこの世界のテクスチャに馴染もうとして拒絶されているかのように。クライは左腕の端末を操作し、情報確認を行った。自分の精神状態に異常はない。だが、心拍数だけが、わずかに上昇していた。
恐怖か？

いや、違う。
指先が震える。それは寒さではない。
次の鍵穴で、何が見えるのか。そして、この少女というバグが、どのように世界を壊していくのか。彼はポケットの中で、小型の記録媒体を指先で弾いた。そこには、まだ何も録音されていない。だが、すぐに満たされるだろう。この街の終わりの音を。

「行こう、レイ」

クライが彼女の名前を呼ぶと、少女は驚いたように目を開けた。
「……知ってたの？」

「君の端末の表示が、一瞬だけ空白になった。それが君の名前だ」

クライは嘘をついた。
ただの直感だ。だが、彼女はそれを真実として受け入れたようだった。
「そう。私はレイ。忘れられた影」

彼女はそう名乗り、クライの袖を少し摘んだ。
その感触は、驚くほど温かかった。データの塊であるはずの彼女に、体温があることが不思議だった。二人は、鍵穴を探して歩き出した。夜の街は、二人の影を飲み込み、すぐに吐き出した。影は長く伸び、やがて一つに重なり合う。
クライは袖の感触を確かめるように、腕を少しだけ強めた。

美化された都市の皮膚の下で、何かが音を立てて腐敗し始めている。
その腐敗の匂いを嗅ぎながら、彼らは夜の深淵へと歩を進めていった。

## 逆視線の律動

男の顔が、まず曖昧になる。
輪郭が溶ける。
それは塗料が肌に重ねられたのではなく、脳の知覚領域へ直接「無」の信号が書き込まれた結果だった。世界からその座標を削除する、強制的な忘却。
処理が始まった。

クライは古びた鉄骨の陰に身を沈め、その光景を眺めていた。
隣には、既にそこに在るかのような静寂を纏った少女が座っている。
レイだ。
淡いグレーのワンピースを着た彼女は、膝を抱え、崩壊していく男性を凝視していた。その銀色の髪は風に揺れることなく、都市の排気熱に含まれる微細な塵と共に沈殿しているように見えた。

「綺麗」

レイの声は、風の音に紛れて消えそうなほど微かだった。
感嘆でも、嘆きでもない。ただの事実確認のような口調だ。

男の左足が、輪郭を失い、背景のコンクリート壁と同化した。
次に右肩。
内臓、思考、記憶、それらすべてが圧縮され、都市の「綺麗さ」を構成するピクセルの一部となっていく。
痛みはないはずだ。システムはそれほどに完璧にできている。脳に直接「快楽」の信号を送りながら、存在そのものを抹消する。
人間的な死ではなく、データ的な上書き。

クライの左腕の端末が、まるで小さな生き物のように脈打ち、皮膚の下で熱を孕んでいる。
美化された都市の皮膚の下、腐敗した傷口を舐め回すように、その熱が走る。
彼は無意識に人差し指を伸ばし、空中のノイズをなぞるように指先を滑らせた。
抹消したい。抹消して、その残骸を確認したい。
汚れたレンズを拭う仕草だ。
虚空に残る触感のない残滓を指腹で確認する。

かつて彼が書き込んだ一行のコードが、ある少女の笑顔をピクセルの残骸へと変えた瞬間を思い出す。
その罪悪感の質量が、指先にのしかかる。
バラバラに分解され、都市の「綺麗さ」を構成するピクセルの一部となっていく。
その崩壊のプロセスに、陶酔と言い難い微かな痺れを感じる。

男の上半身が半分ほど消化された時だった。
クライの視界が、予期せぬ違和感に引っ掻かれた。

通常、美化プロセス中の対象は、自我を保たない。
脳機能は麻痺し、ただの肉の塊として処理される。それはシステムの仕様として保証されている。
だが、男の目が動いた。
上書きされかけた瞳が、焦点を結ぶ。
その視線は、自分を食らっているシステムなどを見ていなかった。
虚空を貫き、レイヤーの裂け目、この鍵穴の向こう側へと、真っ直ぐに伸びていた。

クライの背筋に、冷たい指が這うような感覚が走った。
見られている。
監視カメラの機械的な視線ではなく、有機的な、湿った視線。
それは、失われた記憶や、切り捨てられた感情たちが、泥の中から再生しようとしているかのようだ。

男の口元が、崩壊する輪郭の中で微かに動いた。言葉は発せられていない。だが、その目が語っていた。
『確認』
『共犯』
『気づいている』

拒絶の叫びでも、助けを求める悲鳴でもない。
まるで、長い間待ち合わせをしていた友人に会った時のような、静かで、あまりにも落ち着いた確認の眼差し。

クライは呼吸を忘れた。
自分がシステムの設計者であるという優越感が、そのたった一瞬の眼差しの前で脆くも崩れ去るような錯覚に襲われた。

レイは、男の目と合っていた。
鍵穴の向こう側から投げかけられたその「逆視線」を、正面から受け止めていた。

「ありがとう」

レイが口にした言葉は、空気の重みを変えた。
それは男に向けられたものか、それともこの残酷なプロセスを可能にしている世界か、あるいは自分自身への言葉か。定かではない。

男の輪郭が、最後の光を失う。

あるいは、データの欠落によって描画されなくなったのか。
残されたのは、何事もなかったかのような無機質な路地裏の風景だけ。

数秒の沈黙が流れた。
都市の背景ノイズだけが、耳障りな音を立てている。

「見ていましたね」

レイが振り返る。その表情は、いつもの無表情に近い穏やかさを取り戻していたが、頬の色だけが不自然に蒼白だった。

「あの人は」

「ああ」

システムのバグか。それとも、美化処理の限界界に存在する「抜け道」か。

レイは手を胸に当てた。心臓の鼓動を確認しているのか、それとも自分がまだここに在るという実感を探っているのか。

クライは端末の表示を確認する。ログには異常なエラーコードは記録されていない。美化プロセスは正常に完了している。

「確認……ですか」

レイはその言葉を口の中で転がすように繰り返した。

そこには、汚れた水たまりがあった。

レイは水たまりの前に歩み寄り、覗き込んだ。
