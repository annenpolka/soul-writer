# 嘘つきたちのシンセサイズ

## 第1章：タグ剥離の儀式

自動ドアが音もなく開いた。

空気が変わる。無機質な、それでいて微かに錆びたような電子の匂い。

わたしは一歩、足を踏み入れる。ここは叔父の所有する実験的MR施設、「エターナル・ループ」。名前が陳腐で、少し吐き気がする。世界中のar_contactが一時的に無効化される、特異点。

中に入ると、視界の端で点滅していた常駐ウィンドウが、ひとつ、またひとつと消えていく。

視界がクリアすぎて目が痛い。

推奨事項も、属性も、剥がれ落ちている。

剥がされた。

デジタルの皮膚を。

部屋の中央に、一人の人影があった。ただの肉の塊として、ここに座っている。生身の人間というのは、これほどに平坦で無防備な顔をしているのか。愛原つるぎ。

「よう、委員長」

彼女は顔を向けなかった。視線の先には何もない。剥き出しのコンクリートの天井だけがある。

「挨拶、古くない？」

わたしは短く返す。

「古いねえ。1984年には、もうそんなの流行ってなかったよ。ああ、もうわかってる？ この空間のこと」

つるぎがようやくこちらを向いた。その瞳は、網膜に投影された偽物の光を失って、生々しい漆黒の瞳孔を露わにしていた。

「タグ、消えてる。死体みたいに見える」

「失礼ね。私は生きてるわよ。生きているからこそ、これからやるの」

彼女は指を鳴らした。音が響くと同時に、わたしの首筋に冷たい風が感じた気がした。物理的な風ではない。

シンクロ・ワイヤー。これで私とあなたを、直結する。

「ねえ、知りたい？ わたしが、あなたのことをどう『認識』してるか。推奨事項（レコメンド）に頼らない、生のデータで」

拒否権はない。

問答無用で、ノイズが脳内に流れ込んでくる。

ギュン、という鈍い痛み。血管の中に鉛を流し込まれたような重圧。

まるで、久しぶりに自分の皮膚に触れたかのような、生々しい感触。

わたしは壁に手をついて体を支える。呼吸が浅い。

「痛い……のは当然。あなたの脳味噌、未開発だから」

つるぎの声が、直接頭蓋骨内に響く。鼓膜を通っていない。

彼女の思考が、垂れ流されている。

渇き。誰かを傷つけたいという衝動。世界を書き換えたいという肥大化した自我。そして、その底に沈殿する、黒いタールのような愛着。

これが、愛原つるぎか。

「×したい、と思ってる」

彼女は笑った。口元だけが歪む。

「あなたのこと。×して、中身を確認したい。あなたがただの『学級委員長』というタグの集合体じゃないことは知ってる。だからこそ、壊したら何が出てくるのか、興味があるの」

わたしは自分の内側を見る。

恐怖はある。だが、それ以上に、奇妙な安堵感があった。

ここには「無関心」という安全地帯がない。彼女の殺意は、濃厚で、粘着質で、それゆえに圧倒的な「実在」としてそこにあった。

「システムが、怒ってるみたい」

つるぎが呟く。

視界の端が明滅し始めた。ノイズが走る。それは叔父の管理プログラム、この空間の「正常」を維持しようとする機能だ。

壁が迫ってくる。

ただの壁ではない。灰色の、感情のない、無限に続く「平均値」の壁。世界中の穏やかな無関心が凝縮されたような、質量のある空虚さ。

「私たちを、訂正しようとしてる。『正常』に戻しちゃおうって」

つるぎの声に、焦燥が混じる。

押し潰される。

わたしたちの「異常」な親密さが、この無機質な壁によって飲み込まれようとしている。肌が薄く削り取られていくような感覚。デジタルの皮膚を剥がされた上に、今度は存在そのものを均質化されようとしている。

逃場はない。

「やっぱりね、こいつら、理解してない」

つるぎが立ち上がった。彼女の思考が、赤熱した鉄のように熱くなるのを感じる。

「愛とは何かを。殺意と表裏一体だってこと」

彼女が手を伸ばしてくる。わたしの首元を掴むように。物理的な手ではなく、データの爪。

「トウコちゃん。私はさ、あなたが×される瞬間が見たいの。それが、私たちが『生きている』証明になると思わない？」

警告音が鳴り響く。推奨事項（レコメンド）のアラート。脳内で『自傷行為を中止してください』『社会的適合性を維持してください』という無機質なテキストが滝のように流れる。

無視する。

つるぎの殺意が、限界まで昇華される。

×す。

×したい。

×してやる。

その衝動が、わたしの中にも共鳴する。

わたしの魂（ほんのう）は確かに危機を感じ取っていたから。

共振する。まるで、自分自身が自分を×そうとしているような、究極の自己破壊欲求と、他者からの破壊願望が混ざり合う。

痛い。

意識が白濁する。

だが、その痛みだけが鮮明だった。

「ああ、そうよ。これよ、これ」

つるぎの声が、歓喜に満ちて震える。

迫りくる無関心の壁に、ヒビが入る。

灰色の壁が、赤黒い閃光によって砕け散る。

彼女の純粋すぎる破壊衝動が、この世界の偽りの平穏を焼き切った。×すという行為は、否定できない強烈な肯定だから。

わたしは膝をつく。床の冷たさが、唯一の現実として肌に伝わる。

「ほら、見て。壁、割れた」

つるぎが囁く。

視界が歪む。灰色の壁が崩れ落ちて、その向こうにある深淵のような闇が覗く。それは「正常」な世界が隠していた、底なしの虚無。

わたしは息を吸い込む。

肺が焼けるように痛い。

「これが、私たちの愛ね」

つるぎが笑い声を上げる。それは乾いた音で、それでいて甘美な響きを持っていた。

わたしは彼女を見上げる。

タグも、推奨事項（レコメンド）も、嘘の皮膚もない。ただの、殺意を抱いた人間と、それを受け止める人間。

「……×して、みる」

わたしの唇が、勝手に動いた。

それは彼女への挑発か、あるいは同意か。言葉になる前の、うごめく衝動。

つるぎの瞳が、さらに鋭く細められる。

「いい返事よ、委員長。これが、愛よ」

私の手に、見えないナイフの柄があった。それは心臓を突き刺すためのものか、世界を切り裂くためのものか。

どちらでもよかった。

ただ、この痛みだけが真実だと、わたしは知っていた。
