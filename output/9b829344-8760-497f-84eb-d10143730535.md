# ノイズ・アンカー

## 幻視の過密

合成ゴムが軋む音が、骨を伝って脳底に届く。視界の左隅、警告灯の点滅。周辺ユーザー密度、5000人。

夕暮れの通学路は閉鎖されている。肌にまとわりつく湿った冷風。5000人分の熱気が、仮想の層として皮膚の上に張り付いている。実際にあるのは冷気だけだ。システムは立ち入りを非推奨と告げる。進路は変えない。視覚ノイズの走る境界線を、アスファルトの軋む音だけを頼りに踏み越える。

右手の親指で中指の爪を押し込む。変形した爪の裏側の白い濁りが、夕陽に透けて見える。その視覚的な歪みだけが、この空虚な風景と自分を繋いでいる。

視界が歪み、路面から鉄筋が穿ち出る。錆びついた金属の塊ではなく、レンダリングされずに視界に露出したエラーだ。鋭利な角ばかりが強調され、夕焼けを刺している。

チャイムが鳴る。終業放送の無限ループ。耳垢を圧迫する音に合わせて視界の端に字幕が流れる。

『工程表、第一段階。外壁解体』
『鉄筋コンクリート含有率、確認』
『有害物質、飛散防止シート設置』

歌詞のように流れる文字列。5000人の合唱団が見えない席に座り、データの消去を讃えている。

「……帰る」

独り言が漏れる。

視線の端にノイズが混じる。輪郭の定まらない人影がアスファルトを滑っている。

「ラララ〜、ラララ〜」

愛原つるぎだ。アカペラで校歌を口ずさむ。歩幅に合わせて制服のデザインがジャージからセーラー服、そして軍服へと切り替わる。レイヤーのズレ。

「おい、透心。随分とごった返しているの。五千の亡霊が、お前の頬に息を吹きかけているかもしれぬがね」

つるぎが近づき、顔の輪郭を指でなぞる。

「……うるさい」

「息苦しくはないか？ だが、あの連中は息をしない。ただの残滓（のこりかす）、それが群れをなして蠢（うごめ）いているのだよ。美しいだろう？」

つるぎが足元の鉄筋を蹴る。音も立てず、ノイズのように揺らいだ。

「お前、その顔……座標データが裂けておるようだ。だがそこに、面白い色が滲んでる」

背中を向ける。靴のつま先を、さらに強く路面の亀裂に擦り付ける。

視界の端で字幕がテンポを上げる。

『第二段階。内部躯体の切断』
『粉塵濃度、許容値オーバー』

チャイムの音色が歪み、リバーブがかかる。

道の先、鉄筋のそばに人影。

音無ハルだ。

露出した骨断端に耳を押し当てている。壊れかけの神経を聴いている。

右手が空中で細かく動く。見えないフェーダー操作。足指が靴の中でリズムを刻んでいる。

「……弦が弾ける音。きれい」

呟きがチャイムのノイズに混じる。

「不協和音か？」

振り返る。瞳は焦点が合っていない。透心の顔ではなく、輪郭の外側の「空気の揺らぎ」を追っている。

「……何をしている」

「チューニングよ。未だ、幕は下りていない」

耳元には骨伝導レシーバー。

「レコメンドが鳴らしているチャイムとは違う。素材の悲鳴そのもの」

指でレシーバーを示す。

「聞く？ 生の音」

ARフィルター越しの音が胃を逆撫でする。圧縮され加工されたクリアさは、空気を希薄にする。頭痛の種。

「……バカなことを言うな」

「聴覚崩壊を恐れているのか」

表情の変化はない。ただ、観察している。

「……もっと、聞かせろ」

思考より先に、言葉が喉を掠れた。

「賛成」

ハルが顎でレシーバーをしゃくる。

顔の輪郭がノイズで揺らぐ。震える手を伸ばし、耳元の小さな黒い端末に触れる。冷たいプラスチック。

指先に力を込める。最大音量へ。

瞬間、聴覚が暴力を伴って侵入してきた。

チャイムではない。歌声でもない。

鉄が鉄を削り取る摩擦音。コンクリートが内側から砕ける爆音。崩れ落ちる重い轟音。

ARによる補正のない、現実のノイズ。

高周波が内臓を直接握りつぶし、低域の振動が骨格を共振させる。世界が傾く。

その音の中にだけ、確かな「重さ」があった。

指を離さない。鼓膜から血が流れているかもしれない。平衡感覚は完全に狂っている。

巨大な単一な振動が空気を押し潰し、皮膚を刺す。5000人の圧力が、固体となってのしかかる。

それでも、この痛みだけが、この場所で自分が生きている唯一の証明だった。

ハルが口を動かす。言葉は聞こえない。その唇の動きは、指揮者がタクトを振り下ろす瞬間に似ていた。

透心は空を見上げた。

美しい夕暮れの色も、警告の文字もない。

ただ、灰色の雲が垂れ込めているだけだった。

## 周波数の共鳴

靴底が擦れる。乾いた音が骨を伝う。

つるぎの背中が揺れる。輪郭が滲んでいる。歩くたびに、セーラー服の襟がジャージの袖に変わり、軍服のボタンが光る。その度、彼女は右手の甲で自分の頬を擦り、レイヤーのズレを確認するよう視線を泳がせる。視覚的な追従が間に合わない。

「そこ、踏むなよ」

つるぎが指差す。先には何もない。空間だけが開けている。

「見えない壁がある。座標データが欠落してるから、システムが『ないもの』として処理してる。ぶつかると脳がショート死するッス」

透心は足を止める。つるぎは、その見えない壁に手をかざす。掌が透ける。ARフィルターが背景の空と同じ色に偽装している。カメレオンのように、背景のノイズを吸収して輪郭を消している。

つるぎは笑う。その笑顔も、一瞬で別の表情にスライドする。

「僕たちが通る道は、システムの裏側。ゲシュタルト崩壊した楽譜の裏紙を歩くようなもんね」

ハルの声が後ろから響く。

透心は振り返る。ハルが空中に指を置いている。何もない虚空を、指揮者のように操っている。焦点は定まっていない。彼の視線は、対象そのものではなく、輪郭の外側数ミリにある「空気の揺らぎ」を追っている。

「空気が、厚い」

透心が言う。

「ああ。多勢の気配が残ってる。まるで満員電車のドアが閉まった直後のような、圧迫感。でも、誰もいない」

透心は視線を戻す。頭の中で、数字が弾けようとする。五千。いや、それはもう正確な数ではない。ただの、粘着質な圧力として皮膚にまとわりついている。

つるぎが歩き出す。透心が続く。ハルが最後尾をついてくる。

足音が響く。タッ、タッ、タッ。

不規則なリズム。

視界の端で、何かが点滅する。

推奨事項（レコメンド）。

警告色の枠が浮かび上がる。

『進行方向を修正してください。エリア外です』

無機質な合成音声。鼓膜の奥で反響する。

透心は無視して足を進める。靴底が、見えない境界線を擦る。

「無駄だよ」

つるぎが肩をすくめる。

「システムは執念深い。安全な帰路に戻そうとする。親切な監獄の看守みたいなもんゲス」

警告音が高くなる。ウィーン、ウィーン、と耳障りな波形が空気を震わせる。

透心は眉を寄せる。頭蓋骨の内側が痺れる。

ハルが立ち止まる。

「うるさい」

短い呟き。

ハルが、両手を耳の元に持っていく。骨伝導レシーバーのボリュームを上げるのではない。指を組み合わせ、輪郭の欠片を拾うような仕草だ。

「この周波数、低すぎる。生活音の下に隠された、制御用の信号音が混ざってる」

ハルが口を開く。

唇がわずかに動く。

最初は、息遣いのような音。

次に、喉の奥で共鳴する低い振動。

「ーーー……」

ハルが口ずさむ。

特定の音程。

それはメロディではない。ただの、持続した波だ。

透心は足を止める。

ハルの口から漏れた音が、空気中の粒子を弾き飛ばすように感じる。

警告音が歪む。

『進行方向を……修……ガガッ……』

システムの声が、ノイズに飲み込まれる。

ハルは目を閉じている。指先が、空中の見えないフェーダーを微細に操作している。音程を、わずかに、ミリ単位でずらしている。

「ここだ」

ハルが言う。

透心の耳に、鋭い痛みが走る。鼓膜が震える。

目の前で、推奨事項のウィンドウがひしゃげる。色が反転し、文字が解像度を失っていく。

『エラー。再接続……シ……』

音が消えた。

システムの警告音が、ハルの口ずさむ音程と干渉し合い、互いの波形を打ち消し合った。

完全な静寂が落ちる。

風の音も、足音も、遠くの車の走行音も、すべてが吸い込まれたような真空。

透心は息を呑む。その音だけが、やけに大きく響く。

つるぎが振り返る。口をパクパクさせている。声が出ていない。音声出力がフリーズしたのだ。

ハルが口を閉じる。

静寂は、一瞬だけ続いた。

次の瞬間、世界が戻ってくる。

だが、何かが違う。

視界の隅にあった警告ウィンドウが消えている。ナビゲーションの矢印も、目的地のマーカーも、すべてが消滅していた。

地図がなくなった。

つるぎが口を半開きにし、わざとらしく欠伸をする。退屈そうに視線を外してから、ようやく口を開く。

「……やったわね。推薦エンジン、完全に沈黙させた。今はもう、どこへ行けばいいかなんて、誰も教えてくれないッス」

透心は、自分の手を見る。掌に表示されていたはずの簡易マップも、ステータスバーも、何も表示されていない。素の皮膚だけがある。

ハルが深呼吸をする。

「綺麗な静寂。システムが鳴らしていた余計な装飾音が消えた。今は、構造体の呼吸音だけ」

ハルが、透心の横に来る。

「聞こえる？ この空の下で、世界が軋む音」

透心は耳を澄ます。

システムの音が消えた分、遠くで響く、何かが崩れるような微かな音が聞こえてきた。鉄が擦れる音。コンクリートが割れる音。

それらが、混ざり合わずに、個別の音として存在している。

「行くよ」

透心が言う。

迷いはない。

目的地の座標データは破棄された。帰路の情報は消えた。

だが、足元の地面は硬い。靴底が擦れる感触がある。

透心は、つるぎの示した方向へ足を踏み出す。

見えない壁の向こう側へ。

ハルが、軽く口笛を吹く。それは、先ほど口ずさんだ周波数とは異なる、軽快なリズムだった。

三人の影が、夕暮れのアスファルトに伸びる。

システムの管理外の領域へ。

透心は、右手の親指で中指の爪を押す。爪の裏側の白い濁りが、視界の端で揺れる。痛みだけが、ここが現実であることを告げている。

歩き出す。

リズムを刻んで。

世界は、まだ終わっていない。ただ、静かになっただけだ。

## 鉄筋の錆びた歌詞

正門は二つあった。

一つは、午後五時の光を透過して半透明に浮遊する、完璧な矩形の記憶。もう一つは、その記憶の下で錆びつき、ねじれ、地面に沈もうとしている鉄骨の死骸。

靴底を擦る。アスファルトの亀裂に引っかかるゴムの抵抗が、脛骨を逆流してくる。ここは境界線だ。過去と現在が、座標の精度を無視して重なり合っている。

「再起動対象区域」

つるぎが、空中に浮かぶ誰にも見えないウィンドウを弾くように指を動かしながら言った。彼女の輪郭が、一瞬、セピア色の粒子に分解される。

「システムが一度、データを消去して書き直そうとする場所。つまり、ここから先は『まだ存在しない』ことになってるわけでゲス」

彼女はわざとらしく欠伸をし、自分の頬のラインを指でなぞった。デジタルな皮膚の感触を確認しているようだ。

「書き直される前の、ゴミ捨て場ってわけか」

答えると同時に、視界の端で何かが弾けた。

文字だ。推奨事項（レコメンド）のテキストが滝のように流れ落ちる。『安全な帰路』『保護者への連絡』『天候予報』。意味が剥がれ落ちた文字列は、まるで寄生する菌糸のように眼球の裏側に張り付く。それは情報ではない。ただの、視覚的な圧力だ。

「綺麗」

ハルが呟いた。

彼は耳元の骨伝導レシーバーを調整し、空中に指を這わせる。まるで、見えない楽譜をなぞるように。

「この空気の密度、最高ね。まるで、何千人もの息遣いが一箇所に凝縮されてるような……重低音が響いてる」

ハルの瞳が、虚ろに開かれる。彼は見ている。私には見えないものを。この空間に滞留する、目に見えない「卒業生たち」の残滓を。あるいは、この世界そのものが崩壊していく過程を、一つの交響曲として聞いているのかもしれない。

耳鳴りがする。チャイムの音色が、限界まで増幅され、もはや音として認識できない領域に達している。それは鼓膜を震わせる振動ではなく、頭蓋骨を直接削り取るような、物理的な暴力となっていた。

「……うるさい」

言葉にする前に、口の中で舌が絡まる。

空気が重い。呼吸するたびに、肺に異物が混入するような錯覚。網膜の中央で光景が干渉し合い、胃液が食道を逆流させる色の混濁が生じる。『×す』『×す』『×す』。殺害を意味する伏字が、無数のテキストの隙間から、ちらつきのように見え隠れする。

ハルが、錆びついた門柱の鉄骨に近づいた。

彼はその表面に耳を押し当てる。まるで、病床の恋人に寄り添うように優しく、そして貪欲に。

「音なのよ、透心。世界が崩れる瞬間の、一番最初の手がかりなの。この鉄が歌ってる……。データが剥がれ落ちて、素の材質が露出していく音……。最高のアコースティックね」

彼は目を細め、陶酔したように微笑む。その指先が、鉄骨の赤錆を愛おしそうに撫でる。汚れた金属を、楽器の響板のように扱っている。

酸っぱい液体が喉の奥込み上がった。

崩壊する世界を前にして、ただ一人だけ、その崩壊の音色を楽しんでいる。彼の求めている「静寂」とは、このようなカオスの果てにあるものなのか。

「おい、透心。つまんないこと言わせないでよ」

つるぎの声が、遠くから響く。彼女の姿が、過去のレイヤーと現在のレイヤーの間で、何度も点滅を繰り返している。

「ここ、足場も何もないデータの墓場なんだぜ。お前が探してる『帰路』なんて、最初からここにはなかったんじゃねぇの？」

無関心な指摘。だが、その言葉が鋭利な刃物となって、思考の隙間に突き刺さる。

帰路がない。それなら、私は何のためにここに立っているのか。

視線が、錆びついた鉄骨に吸い寄せられる。

それは、かつて正門の一部だったものだろう。塗装は剥がれ落ち、赤茶色の酸化物に覆われている。歪んだ形状。鋭利な切断面。

それが、唯一の「確か」なものに見えた。

浮遊する文字列も、増幅された振動も、重苦しい空気の密度も、すべてが嘘めいた幻覚に思える。この鉄骨だけが、私の目の前に、物理的な質量を持って存在している。

私は歩み寄った。

靴底が、瓦礫を踏む音を立てる。

右手を伸ばす。

親指で爪を押す癖が動き出そうになるが、それは止めた。今回は、もっと直接的な接触が必要だ。

掌全体で、鉄骨の表面に強く押し込む。

冷たい。

その瞬間、死んだ物体特有の冷気が血管を逆流して心臓へと突き刺さった。

摩擦係数が掌の皮膚を削り取るような鈍い抵抗。ザラザラとした錆の感触が、掌の紋理に食い込む。錆の臭いが鼻孔を突く。鉄の味が舌の先にまで上行する。微細な凹凸。硬質な圧力。

力を込める。

鉄骨は微塵も動かない。私の力など、この物体の存在意義にとっては無に等しい。それはただ、そこに在り続ける。私の認識や、システムのデータなど関係なく、絶対的な物質としてそこに居座っている。

指先が、錆の粒に削られていく。鈍い痛みが走る。

鉄の冷たさだけが、私の体温を奪いながら、私をこの世界に繋ぎ止めていた。

「……現実だ」

呟きが、乾いた音を立てて消えた。

鉄骨は冷たい。硬い。そして、痛い。

それは、ARフィルターが生成するどんな美しい映像よりも、どんな快適な推奨事項よりも、鮮烈な真実だった。

ハルが、こちらを振り返る。その瞳には、鉄骨に触れる私の姿が、どう映っているのだろう。

「あなた、その手……」

ハルが言いかけて、口をつぐむ。彼は空中のフェーダーを操作するように指を動かし、何かを調整した。

「構わないわ。その音も、この場所の一部だから」

つるぎが、鼻で笑う。

「チッ、感傷家は胃が痛くなるんでゲス。早く進むよ、この先に何があるか見ものだし」

彼女は踵を返し、再起動区域の奥へと歩き出した。その背中が、ノイズに揺らぐ。

私は、鉄骨から手を離した。
