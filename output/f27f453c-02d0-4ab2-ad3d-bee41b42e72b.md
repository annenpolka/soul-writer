# 重力を喰らう朝、タグのない街で

## 0度の傾斜と信号待ちのパレード

街灯の光がアスファルトに落ちているはずの位置に、何もない。

いや、正確には光はある。オレンジ色のナトリウムランプの円形が、ぬれた路面に重なって鈍く光っている。だが、視界の端でいつも浮遊していた青白い文字の列、通行人の頭上に張り付く属性タグ、そして歩幅に合わせて点滅する帰宅ルートの誘導ラインが、ことごとく消えている。

ただの街だ。ただの、コンクリートと塗装された金属の塊。

……はぁ。面倒だ。

俺は立ち止まり、誰もいない交差点の中心で、空を見上げる。そこには雲の流れる速度を示す数値も、大気汚染の指数も表示されていない。ただ、黒い空に、灰色の雲が流れているだけだ。まるで、何十年も前の安っぽいフィルム映画のワンシーンを見せられているようだ。

頬の肉を爪の先で強く掴み上げる。皮下組織の柔らかさと、爪の硬さを確かめるように指に力を込めて揺らす。

鋭い痛みが走る。視界が一瞬だけ白く弾け、ノイズが混じる。

ああ、これだ。

監視の目から解き放たれた安堵と、自分という実体が消去されていく恐怖の間で、揺れ動く胃袋の重みだけが、今の俺をここに繋ぎ留めている。麻酔が効き始めた時のような浮遊感なんて胡散臭い。爪が食い込む熱だけが、デジタルの嘘で塗り固められた世界において、俺という有機的なエラーの座標軸になってくれる。

誰もいない空間に向かって、背筋をピーンと伸ばす。右足のかかとを揃えて、型どおりの完璧な敬礼を行う。

袖口が擦れる乾いた音が、夜気に吸い込まれる。

……ふっ。

滑稽だ。俺は喉の奥から乾いた笑い声を漏らし、頬に残る爪の跡を指でなぞる。

歩き出す。

街全体が、どこか穏やかな下り坂のような錯覚に支配されている。あるいは、街路のアスファルトが、まるで巨大な滑り台のように、前方へと緩やかに、しかし確実に傾斜している気がした。前方を歩く人々が、ただの肉の塊として、互いに接触を避けようともせず、同じ速度で流れていく。

不思議と、その奇妙な歩調が気持ちいい。

靴底が路面を擦る音が、遅れて聞こえてくる。俺はまた、制服のリストバンドを指で弾いた。ゴムが皮膚を叩く軽快な打撃音と、手首に走る微かな痛み。このリズムが、世界から徐々に剥がれ落ちていく俺の意識を、無理やり繋ぎ止めている。

視界の端で、点滅するアイコンが気になる。

推奨事項。システムからの行動誘導だ。今のそれは異様な頻度で明滅し、まるでてんかん発作を起こした蛍のようだ。無視する。耳元のデバイスを指で押し込み、物理的な音を遮断する。

今の俺に必要なのは、正解への誘導なんかじゃない。この見えない傾斜に、身体を委ねることだけだ。

ふと、脇の路地裏を見る。そこは、街灯の光も届かない、闇が濃く溜まり込んでいる場所。システムの管理地図上では、灰色の空白として処理されるエリアだ。通常のナビゲーションなら、絶対に選択しないルート。

俺は、効率的な帰宅ルートを示す矢印が指し示す大通りを背にした。

傾斜に従うように、路地裏へと足を踏み入れる。

冷たい湿気が、肌を撫でる。

ここから先は、俺の足という信頼できない計測機器だけが頼りだ。

モニターの青白い光が、網膜を焼き付くように照らしている。

コーヒー、冷めた。

薄汚れた白衣のポケットから、缶コーヒーを取り出し、一口飲む。液体は、もはや味覚を刺激する温度ではなく、ただの物理的な冷たさとして食道を滑り落ちていく。

……か。なるほど。温度変化のログだけが、正確に残ってる。

目の前のメインコンソールに、赤と黄色の警告灯が滅茶苦茶に明滅している。システム修復を促すアラート音が、うるさいほどの音量で鳴り響いている。

『AR同期率、低下』

トン、トン、カ、トン、トン。

その代わり、モニターに映し出されるノイズだらけの映像に目を細めた。

綺麗だ。

都市の輪郭を形作っていたレンダリングが、沈泥のように軟化し、地面へと垂れ下がっている。嘘の皮膚の下から骨が見えたわけではない。むしろ、都市全体が深い息を吐いて、コンクリートの質感が物理的な重みを持って視界を圧迫し始めている。歪んでいて、解像度が低くて、使い物にならないほどの崩壊。

だが、その濁りこそが、この都市のありのままの「正体」ではないか。

私はキーボードに手を置き、エラーログの羅列を詩的な日誌として書き殴り始めた。

「零時、街は錯覚を吐き出す。ビルの角が滲み、信号が溶ける。それは病気ではなく、退化した文明の夢遊病だ」

眼球表面が乾燥して、モニターの粒子が滲んで一つの巨大な有機的な模様に変化していく。視界全体が、ノイズに侵食されてくるのを待つ。硝子が網膜に焼き付いて、景色が私の方へ侵食してくる。

この監視室は、世界が終わるのを特等席で眺めるための棺桶だ。私は、修復の責任者としての義務よりも、この崩壊の瞬間を独占したいという欲求の方が、遥かに強いことに気づいている。

私はまた、デスクを叩いた。

君、叫んでるのに、音が足りない。だから私が足してあげる。

リズムが、夜の静寂の中で響く。

信号、変わらない。

交差点の向こう側、信号機のレンズが滲んでいる。赤色が、アスファルトの熱で溶けたように周囲に滲み出し、境界線を曖昧にしている。

きれい。

背景が溶けてるところ、すき。

視界の端に、ビルの輪郭が追いかけてくるように滲んでいる。ARの輪郭強調が解除されたせいだろうか。コンクリートの質感が剥がれ落ち、世界が低解像度のテープのように走っている。

これは、父と一緒に観た古いビデオテープの砂嵐に似ている。

画面がノイズに飲み込まれ、音声が途切れ、それでも映像だけが続いているあの時間。懐かしい。胸の奥が温まるのではない。デバイスの冷たさが指先から血管へと逆流し、その冷えが胃袋の底で、父の記憶と反応して重く沈殿していく。

焦点を定めないまま目を細める。

世界がドロドロに溶けていく視覚的めまいが、まるで水の中にいるような心地よい浮遊感として知覚される。

「砂みたい……」

私の声は、低く、遠くで響くように自分の耳に届いた。

信号の色認識が遅れているのがわかる。横断歩道の歩行者用灯器が、青に変わったはずなのに、私の脳内ではまだ赤色が残像として焼き付いている。でも、足は動かさない。

あえて、ノイズの走る路肩に立ちふさがる。

ここなら、誰にも見つからない。社会という名の大きなシステムから、ドロップアウトした夢の中にいるみたい。

不意に、視界が揺らぐ。

街全体が、まるで巨大な船が傾いたかのように、ゆっくりと、しかし確実に前のめりに傾いているのが見える。

歩いている人たちが、皆、同じ角度で身体を傾けて歩いている。無意識のパレード。誰もが、見えない重力に従って、どこかへと流されていく。

私も、その流れに身を任せればいいのかもしれない。

古いデバイスの画面には、何も表示されていない。ただの黒い鏡だ。だが、その中に映る自分の顔は、少しだけ滲んでいて、誰か別の人のようにも見える。

「触れれるかな……」

私は虚空に手を伸ばす。

指先が、向こう側の滲んだ色に染まるのが見えるだけ。でも、その虚無の中に触れている感覚。温かいのは、きっと気のせいじゃない。

信号が点滅してる。まるで肺が動いてるみたい。

私は、そのリズムに合わせて呼吸を整える。

傾いた世界の中で、私だけが静止しているわけでもなく、流されているわけでもない。ただ、ここに溶けている。

路地裏の空気は、ひどく冷たくて、それでいて湿っていた。

大通りの明かりは遠ざかり、ここでは月明かりと、非常口の誘導灯の緑色だけが、世界を照らしている。

足元の感覚が、おかしい。

確かに地面はある。だが、歩くたびに、身体まるで坂道を駆け下りているような、心地よい浮遊感が身体を包み込む。この街は、物理的にもう傾いているのかもしれない。あるいは、俺の三半規管だけが、世界の終わりに合わせて狂い始めたのか。

どちらでもいい。

俺はまた、頬の肉を爪の先で強く掴み上げた。

さっきと同じ場所。だが、今回は爪を深く立て、肉をねじり込むように力を込める。鈍く、重く、長く続く痛み。これが、内部で溶解しそうな自我を強制的に再構築してくれる。

機械的なアラート音を打ち消すような、リズミカルな打撃音。

俺は無意識に、背筋を伸ばし、壁に向かって敬礼をしようとして、その手を止めた。誰もいない。監視カメラもない。いるのは、音の主だけだ。
