# 静寂の逆位相

## 第一章：凍てつく帯域

深夜3時の店内は、重力を失った真空に沈んでいる。空調の送風音すら吸い込まれ、都市全体を覆う静寂化プロトコルの重低音だけが、鼓膜の裏側を直接這い回る。母親の子守唄よりも滑かな、皮膚にへばりつくような安らぎだ。

テーブルの上、白磁のカップから立ち昇る湯気だけが緯度を持って揺れていた。

敬はプレスされたシャツの袖口を整え、カップの取っ手に人差し指を添える。冷たくて硬い磁器の感触を指腹で確認する。0.5秒の静止。硬さが一定であることを保証してから、ようやく液体を口に運ぶ。抽出から正確に5分。温度オーバーしたはずだ。

対面の少女が、フォークを落とす。

カツン、という乾いた音が、静寂の薄膜を突き破って響く。

水瀬莉奈の蒼白な指先は空を切っている。金属が皿の上でリズミカルに跳ねる動きは恐怖による震えではない。通信帯域の応答速度をテストするピン信号のように、正確な周期で往復していた。

敬は視線を彼女の瞳に向けない。焦点は眉間から数センチ手前、そこに浮かぶARレイヤーのグリッドラインに固定されている。彼女の頭上で点滅する赤色の警告インジケーターを視野の端で拾う。

莉奈が頬に手を伸ばす。汗を拭う仕草に見えるが、指先は耳元のポート付近を掠めている。乾いた爪が擦れ合う音。

「……混雑してる」

声はノイズ混じりの放送。

「パケットが、たくさん来すぎて……バッファが、溢れちゃう」

彼女は見えないキーボードを打つ。細い指が律動的に震え、虫の羽が擦れ合うような音が空間を抉る。その微細な振動が天板を伝い、敬がカップを握る指先へと届く。掌に走る微弱な電流のような痺れだ。

敬は呼吸を停止する。肋骨が肺を圧迫するまでの数秒間、肺胞内の酸素濃度を一定に保つことで、自身の生体ノイズを遮断する。指先に力が入り、爪の白い圧力が浮き出る。

「……入力量を、制限すればいい」

敬の言葉は、莉奈の呟きから数百ミリ秒遅れて落下した。穏やかで丁寧だが、温度のない事実確認。

「君の震えは、物理演算エンジンのラグだ。処理落ちしてるだけ」

莉奈がフォークを握り直す。金属が皿を削る不快な音と共に、彼女の周囲の空気がわずかに歪んだ。
無機質な壁面に白い結晶ノイズが走る。ARレンダリングエラー、あるいは「自己触発的な過冷却現象」。視覚化された霜が彼女の輪郭を縁取り始める。
テーブルの水出しコーヒーのグラス。表面の結露が無数の水滴となって伝っている。現実の冷気と、仮想現象のマッピングが奇妙な共鳴を起こしていた。

莉奈は両手をグラスに添える。指先の規則的な振動が水面に微細な波紋を描く。

「見て。これ、メモリリークしてるみたい」

水滴が指先から滑り落ちる。

「drip, drip……って。私のリソースが、垂れていくだけ」

焦点の合わない瞳。恐怖の色はない。そこにあるのは広大なネットワークと合一する貪欲な渇望。
敬は顔見上げず、頭上のアトリビュートタグを追う。警告（アラート）から観測対象（オブザベーション）へ書き換わるテキスト文字列。その崩壊プロセスが、美しい最適化として網膜に焼き付く。

カウンターの奥で、店員らしき影が動く気配がしたが、ここまでが管理されたエリアだ。推奨事項（レコメンド）は表示されていない。

「Kei」

「ここが、もっと広いの。都市全体が、私の……クラスターみたいに」

彼女は靴の中で足指を強く絡ませる。革と布が擦れ合う内なる圧力が、身体機能への過負荷を示唆する。
同期を取るように、敬は右手の指をテーブルの角に押し当てる。冷たく滑らかな平面の感触。莉奈が作り出す圧力と同調するかのように、指先に強い荷重をかける。

「……同期を、とろうか」

敬は立ち上がった。
椅子が床を擦る音は、静寂化プロトコルによって即座に吸音され、鈍い衝撃へと変わる。

彼は莉奈の向かいへ歩み寄る。莉奈は視線を上げず、震える手を自分の胸元へと引き寄せていた。首筋のポートを、こじ開けるように触れている。

敬は彼女の手首を掴んだ。

慰めでも、安堵を与えるための接触でもない。
彼は莉奈の左手腕を、机の上に固定する。親指を彼女の橈骨動脈に押し当てる。
骨が軋むほどの強い圧力。

莉奈の手首の中を、熱い血液が脈打っている。そのリズムは、先ほどのフォークの震えと同じ周波数だった。不規則な乱れではなく、正確なクロック信号に従ったリズム。

「……脈拍数、120。変動幅、ゼロ」

彼女の心拍が、自分の指先を通して直接伝わってくる。生身の温もり。不安定で、予測不可能で、保存効率の悪いデータの塊。
それを「測定」しているという事実が、敬の喉の奥で乾いた喜びとなって膨らむ。

莉奈は痛みに顔を歪めなかった。彼女は敬の手首を見つめ返し、その瞳孔が開いていくのを感じる。

「……帯域が、広がる」

彼女は唸るように呟いた。絶望的な悲鳴ではない。長い渇望の果てにようやく解放を得たような、甘美な安堵を含んだ唸り声。
敬の指先が押し付ける場所から、莉奈の皮膚の下を赤い熱流が奔るのがわかる。

敬の手は震えていなかった。完全に静止していた。まるで手術用のメスや、校正された計測器のように。

「君の震えは、正確だ」

敬は彼女の顔の中心、視線の交差する点よりも数センチ手前に浮かぶ、仮想のグリッドラインを凝視したまま言った。

「……美しい波形」

莉奈の周囲に視覚化されていた霜が、一瞬にして加速し、彼女の身体を白いノイズの殻のように包み込んだ。ARのレンダリングが、彼女の存在を現実から切り離していく。

デジタルな冷気が、敬の指先を侵食した。それは錯覚ではなかった。熱量感知のエラーだ。指先が凍傷のように痺れ、血管内を氷水が逆流する。肉体がデータへと変換される摩擦熱が、神経を焼く。

それでも、敬は手首を離さない。脈拍の周波数を計測し続ける。彼女が「人間」としての境界線を喪失していくその瞬間を、彼は最も精巧な精度で記録していた。

生身の生が、データへと上書きされる音がする。
それは、静寂化プロトコルの駆動音と重なり合い、世界で最も完全な秩序の音として響いた。

敬は、ゆっくりと瞬きをした。
視界を走査線が一本横切り、網膜上の映像を静止画として固定する。記録完了。

## 第二章：空白のメニュー

テーブルに置かれた右手が、マイクロ秒単位でジャギっている。
震えではない。フレームレートの低下だ。1秒間に60回駆動していた物理演算が、5回程度へと落ち込んでいる。コマ送り映像のような、カクカクとした動作。スプーンが重力に従い倒れそうになるが、衝突判定が間に合わず、傾いた角度で空中に静止したまま見える。

敬は左手首のベルトを爪で引き剥がした。硬質なプラスチックの留め具が肌に食い込み、鋭い痛覚が走る。これだけが現実のアンカーだ。目の前の崩壊現象は、単なるビデオ出力のエラーである。そう定義し、思考を固定する。

壁面の木目調のテクスチャが、均一なベージュのピクセルへと塗り潰されていく。窓の外に広がる夜景の光点も、星芒レンズの効果を失い、ただの四角いドットの集合体へと退化した。世界がローポリゴン化していく。

対面に座る少女、水瀬莉奈の輪郭もまた、アンチエイリアス処理が甘くなり始めていた。
彼女の頬のラインが、背景の空気と混ざり合い、滲んで見える。輝度補正が追いついていないのだ。敬は瞬きをせず、その滲みを観測した。焦点を合わせようと瞳孔を開くが、彼女の存在は光学像として網膜に結ばれるのではなく、視界の最前面に合成されるARレイヤーの一部のように振る舞っている。

「……Kei」

莉奈が口を開いた。声帯の振動が空気を伝えるより早く、音声波形が視界の下部にスクロールする。だが、そのテキストボックスは文字化けを起こしていた。彼女の発話内容が、システム側で正しくデコードされていない。

莉奈が空中に見えないキーボードを操作するように指を動かした。だがその指先は、空中を滑るのではなく、空間そのものを引き裂くようにグリッチを走らせた。指の通過跡に、一瞬だけRGBのノイズが残像として焼き付く。

敬は視線を彼女の顔から、頭上の空間へと移した。
そこには、彼女の個体を識別するための属性（アトリビュート）タグが浮かんでいるはずだった。

[水瀬 莉奈 / Student / ID: 774-...]
[孤児化データ（Orphan）。親ディレクトリを失い、参照先を失った幽霊データ]

文字列が点滅している。システムが彼女を「在るもの」として索引できていない。
敬は右手の指先をテーブルの角に押し当てた。冷たく、滑らかなエッジの感触。これが現実だ。

「君のインデックスが、見つからないね」

敬の声は、自分の耳にも他人の声のように響いた。感情の起伏を伴わない、平坦なトーン。

莉奈が顔を上げる。その動作もまた、数フレーム分のスキップを含んでいた。
「私、どこ？」

「ここにいる。座標は正しい。でも、名前が参照できない」

敬は視界の端に表示される警告ウィンドウに意識を向けた。

[推奨事項（レコメンド）：対象なし]
[対象のステータスを確認できません]

連続するポップアップ。システムは、彼女を「顧客」として認識することを放棄している。不要なデータパケットとして、メインメモリから解放しようとしているのだ。

莉奈がテーブルの上に身を乗り出す。彼女の身体が、テーブルの物理境界を無視してめり込んだ。コリジョン判定の破綻だ。彼女の胸元が、無造作に置かれたカトラリーセットと重なり合い、互いのテクスチャを干渉させ合って醜いモアレ模様を描く。

彼女は首筋を親指で強く押した。首の後ろにある、接続用ポートのカバーを引き剥がすような、痛々しい仕草だ。
「Kei、見て。ここ、熱い」

「ただのレンダリングエラーだ。無視していい」

敬は視界の中央をタップし、警告ウィンドウを一つずつ閉じていく。指先が画面を叩く感触はない。だが、脳内ではウィンドウが閉じる「音」が再生される。乾いた、紙が擦れるようなような、電子的な消去音。

莉奈が苦しげに顔を歪める。その表情の変化も、スローモーションのように遅延していた。
「違う……これ、違う。私、食べたいの」

「何を？」

「鉄。そう、鉄の味」

彼女の言葉が、唐突に具体性を帯びる。
敬は手を止めた。警告ウィンドウが一つ、視界の隅に残る。

彼女は「鉄」を求めている。だが、今の彼女には味覚を処理するインターフェースすら残っていないはずだ。なのに、その要求はシステムのエラーログとは異なる、古いキャッシュデータから呼び出されたかのような鮮明さを持っていた。

敬はメニュー画面を起動した。
通常、この時間帯、システムはユーザーの行動履歴に基づいて最適な提案を行う。だが、今の莉奈に対してシステムが提示できるのは「推奨対象なし」という空白のみ。

だが、敬は違った。彼は観測者だ。システムが捨てたゴミの中に、まだ価値のあるデータが残っている可能性を信じている。

彼はメニューのソースコードを展開した。表面のUIではなく、裏側に蓄積されたログ領域へと潜る。
キャッシュの最奥層。アクセス権限のない、忘却の領域。

そこには、無数のテキストデータが渦巻いていた。過去の注文履歴、検索履歴、未処理のクエリ。その海の中から、敬は一つの文字列を探り当てた。

[Iron Supplement Drink / 2019-11-14 / Order ID: 9921]

日付は三年前。彼女がまだ、完全にこのシステムに依存する前の記録だ。

「鉄分剤飲料？」

敬が呟くと、莉奈の瞳が、カクカクと大きく見開かれた。

「そう……それ。渋い、舌に残る、味」

彼女の反応が、一瞬だけ加速した。フレームレートが正常に戻ったかのような、鮮烈な輝き。

敬はそのログデータを詳細に解析し始めた。画像データは破損しており、ノイズだらけだ。だが、成分表示のテキストは読み取れた。

水、果糖ぶどう糖液、還元鉄、デキストリン。

乾燥した物質の羅列。それが「味」という主観的な体験を構成していた。敬はそのリストを指でなぞるようにスクロールした。彼女が求めているのは、この物質の組み合わせなのか。それとも、このログデータが指し示す「過去の時間」なのか。

莉奈が虚空のアイテムを摘まむような仕草をした。何もない空間を指で掴み、口へと運ぶ。実際に何かを食べているわけではない。彼女の唇は閉じられ、喉が動くだけだ。だが、その表情は虚空の不味さを味わっているかのように緩んでいった。

「Kei、お願い。……そのデータ、私に」

莉奈の瞳が、ARのグリッドを超えて、敬を真っ直ぐに見据えていた。焦点が合っている。ノイズが消えている。

だが、それは一瞬の閃光だった。

視界が赤く染まる。

[警告：対象の生体反応が閾値を超えました]
[推奨事項：接続を切断してください]
[対象の精神汚染度：危険レベル]

システムが、彼女を「ウイルス」として認識した。排除対象として特定した。

敬の手首を握る莉奈の手が、急激に透け始めた。指先が光の粒子となって分解され、空中に拡散していく。物理的な接触が、データの干渉へと変化する。

「Kei……！」

莉奈の声が、歪んでいく。電子的な変調がかかり、人間の声帯の音色を失っていく。

敬は右手をポケットから引き抜いた。

彼女を救うために手を伸ばしたのではない。彼女の崩壊を止めるためでもない。

彼は親指と人差し指で、たった一つの金属片を摘んでいた。隠し持っていた「同期（シンクロ）キー」だ。

敬は視界の中央に広がる赤い警告ウィンドウを無視し、莉奈が痛みに押さえていた首筋のポートへと、その鍵をねじ込んだ。

硬質な接続音が、脳髄に直接響く。

これは肉体的な痛みではなかった。圧倒的なデータの奔流が、莉奈の自我を物理的な意識から剥離し、都市という巨大な記録媒体の一部へと書き換えていく。都市の静寂化プロセスが、彼女を飲み込んでいる。

敬は短く告げた。

「アーカイブ、開始」

彼は視界の端を確認した。

[推奨事項（レコメンド）：対象なし]
[同期完了：都市メディアへ保存されました]

彼はそのスプーンを指で弾いた。

乾いた音が、低解像度の店内に響き渡る。音は、解像度の落ちた壁に吸収され、すぐに消えていった。

彼は観測を続ける。彼女が都市の一部となって蒸発した今、それが彼に許された、唯一の行動だからだ。

## 第三章：静寂の逆回生

店内の照明灯が、わずかに周期をずして明滅した。

それは故障ではない。同期エラーだ。神崎敬は、テーブルの向こうに座る少女の輪郭が、周囲の空間からノイズを吸収してぼやけていくのを感じ取った。

水瀬莉奈は、自分の頬をかきむしっている。その指先は白く、力が入りすぎて関節が逆反りしている。だが、その動きに恐怖による躊躇はない。一定のリズムで、機械的に、まるで壊れたレコードを再生しているように繰り返される。

敬はマグカップの把手（取っ手）を親指で強く押し込んだ。磁器の硬さが爪の裏に食い込む感触を確かめる。彼女の異常な熱量に対し、自分の体温この冷めた物体だけが、物理的な現実性を保っているアンカーだ。

「漏れてる」

莉奈が口を開く。声が掠れ、音声パケットが欠落して不安定に揺れる。トーンに悲鳴のような高低差がない。ただのエラーログの朗读だ。
