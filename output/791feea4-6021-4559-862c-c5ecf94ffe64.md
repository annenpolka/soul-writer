# 非同期の生体遅延

## 第1章：視覚停止の余白

午後のオフィスは、死んだ水槽の底を思わせる静寂だ。黒い画面でカーソルが明滅するたび、思考の端が削り取られていく。親指の腹を人差し指の第一関節に押し付ける。骨の硬さと皮膚のわずかなたるみを確かめるだけの、無意味な繰り返し。指先から伝わる固い感触だけが、現実の境界線を主張していた。目に見えない障壁を確認し、その手をデスクの上のファイルに滑らせる。

窓の外、夕暮れの空に奇妙な雲が流れる。単なる水蒸気の塊とも言えない。都市を司るプログラムが更新される際、溢れ出した情報の残骸、漂流する ID タグの群れが作る光の帯だ。あの雲が、都市の記憶を上書きしようとしている最中なのだとわかる。私のアイデンティティも、いずれは物理的な器を捨ててあの光の流れに同化してしまうのか。思考がそこまで及んだ瞬間、脳内でスイッチが切り替わる。顔を窓から背け、再び画面上の点滅へと意識を戻す。

突然、周囲が真っ白に染まる。
通知も警告音もない。景色のレンダリングが停止した。街の雑踏、AR 広告の色彩、通行人のタグ、すべてがノイズに飲み込まれる。数秒間だけ、都市の計算が停止し、物理的な空間へと還る。

息を止める。耳の奥で、鼓動だけが不協和音を奏でている。

交差点の中央で、足が止まる。
白い世界の中で、一人の女性が立っていた。サワだ。
足元には、AR 誘導線も歩行推奨エリアの壁も存在しない。灰色のアスファルトが露呈しているだけだ。右手に持ったサンドイッチを、あろうことか彼女は地面に落とした。

包装紙が重力に引かれ、落ちていく。
だが、何かが噛み合っていない。
私の知覚と、物体の落下速度が一致しない。
「落下」と判定され、物理演算が終了するその直前の隙間。包装紙は空中に静止し、緩やかに傾きを変えている。
窓の外で見た ID タグの雲が、この物体の周りにだけ特異な引力を結びつけているような、そのような錯覚。

喉が渇き、エラーの予感がする。処理落ちだ。完全に同期し、決定論的に全てを固定する「死」の直前に現れる、人間的な意志の痕跡。
コーヒーカップに手を伸ばし、その縁に触れてみる。温かいが、適温を三秒ほど過ぎているようだ。表示される「適温」の数値と、決して埋まらない溝。その程度のズレが、悪くない。

色が戻り始める。白いノイズが彩度を取り戻し、街の風景が再構築される。補強された視覚情報が、剥がれかけた皮膚のように表面を覆う。
サワは落ちたサンドイッチを見下ろしていない。その焦点は、私の肩越しの空間、あるいは耳の後ろの「何もない空間」に固定されている。

「......あの子、風に乗った」

焦点の定まらない響きだ。対面の空間越しに聞こえてくるような。

「......それ、食べるとお腹壊すよ。中身が......空っぽだから」

彼女のつぶやきが、空の裂け目へ吸い込まれていく。
足先が、地面に着く寸前にもう一度止まる。浮遊感を伴う不自然なステップ。重力が一瞬だけ抜け、身体が空気の流体へと沈む。自動補正や予測が及ばない、物理演算の盲点。彼女はそこに自身の身体を「引っ掛けて」いる。

自動削除のプログラムが起動し、赤い警告枠が物体を囲む。
『推奨事項』：不要なゴミの排除。路面の清潔保持。
端の方で、警告ウィンドウが点滅する。

「おい、何をしている」

冷ややかな声が背後から響く。
御鐘透心だ。折り目正しい制服に、一切のしわはない。レンズには最新の情報が流れている。

「その『遅延』の原因が何であれ、無意味だ。従わないデータは、存在価値がゼロだ。消去しろ」

透心の言葉は短く、鋭利な刃物だ。
彼は両手を下ろしたまま、指先をわずかに内側に曲げる動作を止めない。まるで目に見えないキーボードを打ち続けているかのような、無駁のない動作。彼にとって、この包装紙の落下遅延は不快なノイズに過ぎない。すべての事象が、機械的な正解へ即座に収束すること。それが彼の渇望する無菌で摩擦のない秩序だ。

「......信号が、変わったのが、遅かった」

呟く。透心の言葉を聞くふりをして、顔を背ける。網膜の奥で光の残像が滲み、都市ノイズのような白い粒子が舞う暗順応の時間。
透心は、私が「遅延」を放置していることを知っている。そして、その行為を否定しようとしている。

「......馬鹿なこと。非同期は、癌だ」

彼は言い放つ。
私の記憶ログや認識データをリアルタイムで書き換えようとしているのだろう。客観的な世界が混乱していても、主観データ上だけは「既に解決済み」という錯覚を得たい。歪んだ充足行動。

過去と現在が、数秒単位で切り替わる断片的なフラッシュバック。

それが消えてしまう。

身体が動き、決断を下す。
削除のプロセスより早く、遅延した包装紙に手を伸ばす。

視界を遮る赤い警告枠を、指で押し退けるように突破する。
空気の密度が変わっている。粘度のような抵抗を感覚に刻み込みながら、手を突き出す。乾いた指先が受ける微かな抵抗と、プラスチックとの接触。硬質な質感が混ざり合う。

肩が走る。筋肉が軋む。
端の方で、透心が驚愕したような表情を作るのが見えるが、それは関係ない。
私の手が、赤い警告枠を突き破り、空中に浮かぶ「エラー」を鷲掴みにした。

だんっ。

手の中に、感触が残る。
生ぬるい。湿っている。サンドイッチの油が、掌に張り付く。
都市上では存在しないはずの『物体』を、物理的に所持した事実。
それは、職務規定違反による検査員資格剥奪のリスクを超える、ある種の背徳的な熱だった。

視界の補正情報が鮮やかになり、街全体が再び動き出す。
通行人の足音が、視覚的な歩みと同期し、リズミカルに響く。
一切は、何事もなかったかのように「正常」へと戻った。

だが、私の手の中には、「間違い」が残っている。
透心が、私の手元を睨みつける。
表情筋を少しも動かさず、ただ冷徹な視線だけを送る。

「......貴様のその手、汚れている。......ノイズで不快感を催す。放棄しろ」

声から感情は消えているが、その実、自己のテリトリーに入り込む不完全性を排除しようとする敵対心が見え隠れする。

サワはもういない。いた場所には、拡張された誘導光だけが、正確に、完璧に引かれている。
計算からこぼれ落ちた自分だけが「ここにいる」という強烈な実在感を得て、彼女は消えたのだろう。

「......俺の記憶では、今の行為は発生していない。貴様が一度報告書を出し直した、という事実だけが保存された。......了解か？」

透心が言い放つ。
主観データ上では、私が包装紙を拾った事実は、すでに「削除」されているのだろう。
不快感を認識から書き換えることで、この現状を維持している。

私は、透心の顔を見ない。
彼の背後にある虚無、あるいはこの空間の構造そのものに意識を固定する。
ガラス越しに、都市の幾何学的なグリッドを透かして見ている。

頷く代わりに視線を落とし、返事にする。

ポケットの中に、包装紙をねじ込む。
掌の下で、その物体が温かい油の滑りと、自身の脈動を重ねて鼓動する。

透心は、鼻で笑うように息を吐き、踵を返した。
その背中が、群衆の中に溶けていく。
完全に同期された背中は、既に死んでいる遺骨に見えた。

ポケットの中のゴミの熱を感じながら、再び歩き出す。
足元のアスファルトが、溶け出している。
あそこだけ、計算から漏れている場所だから。
視界に浮かぶ誘導線に対して、常に数センチ外れた位置を歩く。
提示された快適な整合性は「偽物」だ。

それとも、このズレたまま、隙間に埋もれてしまうのだろうか。

## 第2章：重力と歩幅の交差点

## 第3章：剥離したIDの雲海

## 第4章：恒久的不整合への帰還

蛍光灯の高周波音が、頭蓋骨の内側をかすめる。白い天井、白いデスク、そして白い光。空間は均一に埋め尽くされているようで、実際には自分の身体だけが鈍い色の影を落としている。カジワラはプラスチックの椅子に沈み込み、画面上の点滅するカーソルを見つめ続けている。ブリンクの間隔は毎秒二回。都市の基本パルスと完全に一致する数値だ。だが、心拍数がその数値よりもわずかに遅れている。泥水の中を歩くような重さというよりは、機械油が粘度を失っていくような、鈍い停滞の感覚。

ディスプレイには、検査データが整然と配列されている。セクター4から9までの領域において、異常値はゼロ。推奨される事項の遵守率は百パーセント。不純物が除去された結体のような空間だ。カジワラはキーボードに指を置く。打鍵する前に、空中で指先を小刻みに震わせ、見えない障壁の感触を確かめる癖がでそうになるのを、指を数ミリ浮かせて抑制する。その動作を行えば、自分が「外側」へ露出してしまうかもしれないからだ。

規則的なヒールの音が近づいてくる。硬質な音が床の継目を正確に踏み抜いてくる。御鐘透心だ。歩行パラメータはメトロノームのように正確だ。カジワラは視線を彼の顔ではなく、肩越しの壁のシミへと固定したまま、背筋を走る緊張を単なる温度変化として処理しようとする。

「報告書、まだか」

透心の声は、デスクの向こうから降ってくる。感情の起伏が欠落した、ただのパケットのような音声だ。
